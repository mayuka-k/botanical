単体

使い方

アコーディオンのタイトルに（ac-s-p）アコーディオンのコンテンツに（ac-s-c）を付けるだけでそれぞれにタグが振り分けられます。

また、「//アコーディオン」のクラス名をサイトにあったクラス名に変更ください

※同じようにtoggleClassで実装しているアコーディオンについては、上のクラス名を付けて//アコーディオンjsを削除してください

※また、初期表示で開いた状態にしたいときは「//初期表示」を削除してください

HTML
					
<h5 class="ac-p"><a href="" class="ac-s-p">アコーディオンタイトル</a></h5>
<div class="ac-c">ここにテキストが入ります。ここにテキストが入ります。ここにテキストが入ります。ここにテキストが入ります。</div>
JQUERY
									
//idとaria-controls,accordion-panelに番号を割り振る
$('.ac-s-p').each(function (i) {
	$(this).attr({
		id: 'accordion-s-tab' + (i + 1),
		'aria-controls': 'accordion-panel' + (i + 1)
	});
	$(this).next('.ac-s-c').attr({
		'aria-labelledby': 'accordion-s-tab' + (i + 1),
		id: 'accordion-panel' + (i + 1)
	});
});

//それぞれにrole属性、aria-selected、aria-expanded、aria-hiddenを付ける
$('.ac-s-p').attr({
	role: 'tab',
	'aria-selected': 'true',
	'aria-expanded': 'true',
});
$('.ac-s-c').attr('aria-hidden', 'false');

//クリックした際のrole属性、aria-selected、aria-expanded、aria-hiddenを変更する
$(".ac-s-p").click(function (e) {
if ($(".ac-s-c").css('display') == 'none') {
	$(this).attr({
		'aria-selected': 'true',
		'aria-expanded': 'true'
	});
	$(this).parent().next(".ac-s-c").attr({
		'aria-hidden': 'false'
	});
} else {
	$(this).attr({
		'aria-selected': 'false',
		'aria-expanded': 'false'
	});
	$(this).parent().next(".ac-s-c").attr({
		'aria-hidden': 'true'
	});
}

//aタグ無効化
e.preventDefault();

});

//初期表示
$('.ac-s-c').css("display", "none");

//アコーディオンjs
$('.ac-s-block h5 a').click(function (e) {
	$(this).parent().next(".ac-s-c").slideToggle();
	$(this).toggleClass("active-ac");
});

スマホメニュー

実装例

http://dev019.coara.or.jp/itou/yufuin/

使い方

スマホメニューのボタンに（sp-p）開くメニューに（sp-c）を付けるだけでそれぞれにタグが振り分けられます。

JQUERY
				
//idとaria-controls,accordion-panelに番号を割り振る
$('.sp-p').attr({
	id: 'menu-tab1',
	'aria-controls': 'menu-panel1'
});
$('.sp-c').attr({
	'aria-labelledby': 'menu-tab1',
	id: 'menu-panel1'
});
// $('.sp-c').addClass("active-menu");

//それぞれにrole属性、aria-selected、aria-expanded、aria-hiddenを付ける
$('.sp-p').attr({
	'aria-selected': 'true',
	'aria-expanded': 'true',
});
$('.sp-c').attr('aria-hidden', 'false');

//クリックした際のrole属性、aria-selected、aria-expanded、aria-hiddenを変更する
$(".sp-p").click(function (e) {
$('.sp-c').toggleClass("active-menu");
if ($(".sp-c").hasClass('active-menu')) {
	$(this).attr({
		'aria-selected': 'true',
		'aria-expanded': 'true'
	});
	$(".sp-c").attr({
		'aria-hidden': 'false'
	});
} else {
	$(this).attr({
		'aria-selected': 'false',
		'aria-expanded': 'false'
	});
	$(".sp-c").attr({
		'aria-hidden': 'true'
	});
}

//aタグ無効化
e.preventDefault();

});

JQUERY
									
$('ul.btn-tab li a').click(function (e) {
	num = $(this).parent('li').index();
	$(this).addClass('active').parent().siblings().children().removeClass('active');
	$(this).parent().parent().next('.tab-contents').children('div').removeClass('active').eq(num).addClass('active');
	e.preventDefault();
});

//idとaria-controls,accordion-panelに番号を割り振る
$('.tab-list li a').each(function (i) {
	$(this).attr({
	id: 'tab' + (i + 1),
	'aria-controls': 'tab-panel' + (i + 1)
	});
});

$('.tab-c > div').each(function (i) {
	$(this).attr({
	'aria-labelledby': 'tab' + (i + 1),
	id: 'tab-panel' + (i + 1)
	});
});

//それぞれにrole属性、aria-selected、aria-expanded、aria-hiddenを付ける
$('.tab-list').attr('role', 'tablist');
$('.tab-list > li > a').attr('role', 'tab');
$('.tab-c > div').attr('role', 'tabpanel');

$('.tab-list > li > a').attr({
	role: 'tab',
	'aria-selected': 'false',
	'aria-expanded': 'false',
});
$('.tab-list > li > a.active').attr({
	role: 'tab',
	'aria-selected': 'true',
	'aria-expanded': 'true',
});

$('.tab-c > div').attr('aria-hidden', 'true');
$('.tab-c > div.active').attr('aria-hidden', 'false');

//クリックした際のrole属性、aria-selected、aria-expanded、aria-hiddenを変更する
$(".tab-list > li > a").click(function (e) {
	$(".tab-list > li > a").attr({
		'aria-selected': 'false',
		'aria-expanded': 'false'
	});
if ($(this).hasClass('active')) {
	$(this).attr({
		'aria-selected': 'true',
		'aria-expanded': 'true'
	});
}

$(".tab-c > div").attr({
	'aria-hidden': 'true'
});

if ($(".tab-c > div").hasClass('active')) {
	$(".tab-c > div.active").attr({
		'aria-hidden': 'false'
	});
}

//aタグ無効化
e.preventDefault();

});
