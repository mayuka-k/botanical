v2.0 β版 リリースノート

試用版リリース日
2023/6/8
試用版リリースバージョン
v2.0 β版

概要

以下の部分を改修済。

  • コーディングテンプレートをアクセシビリティのみに統一
  • swiper
  • インデント
  • 改行コードの統一
  • 折り返し設定
  • SPフォントサイズの仕様変更
  • _variable.scssの変数の整理
  • slider.scssの整理
  • fontawesomeの使用廃止
  • 表示・非表示のクラス整理
  • ヘッダー
  • フッター
  • トップページ
  • element
  • module
  • formparts
  • form-search
  • slider.scss
  • wysiwyg
  • その他各ページについて

目的

さらに効率よくコーディングを進めるための改良。
今回、開発との連携についても考えながら改修を行う。

リリース内容

コーディングテンプレートをアクセシビリティのみに統一

こちらのテンプレートの更新を廃止し、本コーディングデータを更新していくものとする。

インデント

全体的にインデントがそろっていなかったため調整済。（タブサイズ4）

swiper

swiperバージョンを最新版（9.3.2）にアップデート。そのため独自に入れていたswiperのcssを削除、公式のものを採用。

改行コードの統一

LFで統一。

折り返し設定

bodyにword-break: break-word;overflow-wrap: break-word;追加。

SPフォントサイズの仕様変更

アクセシビリティでNGのため、vwとvminを使うこと自体を禁止。

_variable.scssの変数の整理

テキストカラーとサイトのベースカラーの変数をわかりやすく整理。また、不要な変数は削除。

slider.scssの整理

  • 「.slider-thumb」の記述箇所が2カ所あるためまとめる
  • コメントアウトの有無がバラバラなので、各種コメントアウトを付ける
  • 「スライダーボタン変更箇所」とコメントアウトを入れて変更しやすく調整
  • 1枚のときのスライダーの仕様調整。

fontawesomeの使用廃止

fontawesome関連のもの削除。今後は画像で対応していく。

表示・非表示のクラス整理

  • .pc-block .pc-block-inline .pc-block-flex
  • .sp-block .sp-block-inline .sp-block-flex

ヘッダー

  • ロゴのwidth属性削除。cssでサイズ指定に変更。
  • SP切り替え時に、JSで要素をcloneする仕様を廃止。
  • メガメニューを画像ありのバージョンに変更。
  • ヘッダーのデフォルト仕様を以下に変更。
    固定にする。下にスクロール時に消える、上にスクロール時に出てくるをデフォルトに変更。
  • headの中の★★を削除。

フッター

  • ページトップの記述を整理。
  • フッター上のmarginは削除。

トップページ

  • メインスライダーとは別にスライダーが複数入ることを想定し、「.main-area .swiper-container」へ調整。
  • imgタグをLazy-loading追加。（必須）

element

  • 「・」「※」リストの追加。
  • 背景の.bg-c.inside　SP時の余白を狭める。

module

  • タブパーツの追加。（.tab-list .tab-wrap .tab-contents）
  • ボタンパーツの右寄せ追加。（.btn-block.right）
  • ボタンパーツの画像は疑似要素で表示する。
  • ページングの右寄せ追加。
  • .sentence-block に下線が入るリンクテキスト、テキストの中央寄せ追加。
  • .sentence-block インデント（.indent）と余白無し（.mb0）を追加。
  • SNSアイコン＋テキスト追加。(.sns-share)
  • 規約パーツ追加。（.scroll-text）
  • 横並びのサムネイルリスト追加。（.side-block .overlap）
  • アコーディオン開閉の「＋」「－」バージョン追加追加。（.toggle-plus）
  • 404、503、メンテナンスで使用するパーツ調整。（.static-wrap）
  • テーブルスクロールの「左右にスクロールします」の仕様を以下に変更。
    inner以下になった時にスクロールする。通常のPCは出てこない。

formparts

  • IEのサポートのコード削除。
  • input・select枠のサイズ調整のクラス追加。（.text-s .text-m .text-l .sp-text-s .sp-text-m .sp-text-l）
  • 同意しますの「必須」　SP時に落ちないように調整。
  • .txt-required削除。strong.txt-requiredの仕様推奨。
  • 確認画面にて、パスワードなど半角英数をたくさん入れたときに折り返すように調整。

wysiwyg

  • 動画部分、titleが重複しているため削除。
  • 「wysiwyg」内にある「frameborder="0"」を削除。（HTML5で廃止になったコードでバリデータでいつもエラーが出るため）
  • wyiswigで画像があるものに対しては、alt名を仮置きいれておく。（micheckerエラー回避のため。[問題あり「このリンク内には読み上げ可能なテキストが存在しないため、アクセシブルではありません。 (href="")]）

その他

  • #contentsにoverflow:hidden追加。
  • プライバシーポリシーの内容の変更。
  • _original.scssの削除。
