コーディングについて

ページを作成したら、必ずクオリティチェックをしてください。

  • 対象ブラウザで検証したか（chrome、firefox、edge、ie）
  • 実機で確認（iphone、android、ipad）
  • コードにエラーや間違いがないか
  • 読み込み速度が速くなるような対策をしているか

チェックツール

HTMLのバリデーション

The W3C Markup Validation Service：https://validator.w3.org/

CSSのバリデーション

The W3C CSS Validation Service：http://jigsaw.w3.org/css-validator/validator.html.ja

CSS LINT：http://csslint.net/

ページスピードのチェック

PageSpeed Insights：https://developers.google.com/speed/pagespeed/insights/?hl=ja

モバイル フレンドリー テスト

モバイル フレンドリー：https://search.google.com/test/mobile-friendly?hl=ja

画像の最適化

JPG圧縮ソフト

Caesium：https://saerasoft.com/caesium/

PNG圧縮ソフト

Pngyu：http://nukesaq88.github.io/Pngyu/ja.html

サーバーへのリクエスト数を減らす

  • できるだけまとめられる外部ファイルは一つにまとめる
  • CSSで表現できるところは画像を使わない
